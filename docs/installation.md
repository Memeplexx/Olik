### Olik is not dependant on any framework, but wrapper libs exist for some frameworks
{: .no_toc }

## If you're not using a framework
```bash
npm install olik
```

## If you're using React
```bash
npm install olik-react
```

## If you're using Angular
```bash
npm install olik-ng
```
